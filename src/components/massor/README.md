# Massor design system

My little design system

## Typography

### Lora Font

The Lora font is available for use in Massor components. Lora is a well-balanced contemporary serif with roots in calligraphy.

**Usage:**

```tsx
// Using the Tailwind utility class
<h1 className="font-lora">Beautiful Heading</h1>

// Combining with other styles
<p className="font-lora text-2xl font-semibold">Elegant text</p>
```

**Available fonts:**

- `font-lora` - Lora (serif, for Massor)
- Default font stack uses Geist Sans and Geist Mono
